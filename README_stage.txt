Stage is a 3D simulation package for factory automation using openGL that runs on the Windows operating system. It was written using Microsoft Visual C++.

To install:
   copy the stage.exe file from the Release folder
   copy the stagelib folder to the same directory as stage.exe

Run stage.exe and open a sample workcell in the stagelib/workcells directory.

More information can be found in the manual.2002.04.04 file.
